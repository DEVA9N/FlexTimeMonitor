Welcome FlexTimeMonitor.

This application helps you to keep track of your working hours by monitoring your computers daily uptime.

Highlights

  • clear and lean user interface
  • editable history
  • supports notes

How does it work?

When the FlexTimeMonitor is started it creates a new history entry for the current workday. But it only keeps the very first entry for that day in the history. That way no computer restarts or are logged. It also stores the shutdown time for each workday. But again only the last shutdown is stored. Same happens when the computer enters or leaves a power saving mode.

What it can't

Although FlexTimeMonitor uses a pretty solid algorithm to detect the daily working hours it can not detect day switches. So if you work night shifts this application is not for you (at least at the moment). Since it tries to save your history whenever the computer shuts down it works under critical conditions where data loss can occur - although very rarely. The risk is especially high for forced shutdowns. In this case you can lose you complete history. Create backups of the history file!

Privacy concerns

This application has been developed with privacy concerns in mind. It only stores the absolute minimum of required data and stores it only locally in your home directory. It does not monitor your activities or mouse movement. The history is located in: '''C:\Users\USERNAME\Documents\Flex Time Monitor\History.xml'''
